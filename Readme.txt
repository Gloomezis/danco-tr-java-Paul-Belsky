[program_name]
Fixed world social network v.0.0.1 (alfa)

[program_info]
Социальная сеть для людей интересующихся велосипедами и велокультурой в целом.Позволяет общаться людям из разных стран на велотематику и околовелосипедные темы.
Особенности: 
1. Поиск пользователей
2. Обмен сообщениями между пользователями личными и приватными
3. Позволяет подписаться на интересуюзих вас людей и просмотреть списки подписанных а так же кто подписан на вас.
4. Просмотр событий людей на которых вы подписанны и которые подписанны на вас
5. Форум, для общения на велотематику
6. Сайт реализован в 2-ух языковых исполнениях (Русский/Английский)

[setup]
1. Установить базу данных с помощью файла db.sql из папки support
2. Импортировать проект java проект в eclipse. 
3. Скачать все зависимости с при помощи maven : socialNetwork->maven->Download Sources
4. Обновить maven project: socialNetwork->maven->Update Project
5. Очистить предыдущую сборку если существует-> socialNetwork->run as->Maven clean
6. Очистить предыдущую сборку если существует-> socialNetwork->run as->Maven build
7. Готовый war файл разместить на сервере.
8. Папку frontend разместить на сервере.
или в папке: Tomcat->conf->server.xml  указать путь к фронтенду<Context docBase="d:/FinalProjectSocialNetwork/frontend" path="/" reloadable="true" />
9. Запускаем сервер
10. Наслаждаемся.

Удачи.
